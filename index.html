What is GraphQL?

Query language for APIs

Data source agnostic

GraphQL is a query language for APIs. In the simplest form. It can be looked at as SQL queries, but it is not a database. In fact, GraphQL doesn't care where you get the data. That way, you specify your sources, in memory, SQL databases, No-SQL databases, even other APIs.

GraphQL in action


					{
					  viewer {
					    login
					    bio
					    organizations(first: 3) {
					      edges {
					        org:node {
					          name
					        }
					      }
					    }
					  }
					}
So to give you a taste of GraphQL, this is what a query for a Github user and his or her first 3 organisations looks like

Let's go back to the query. One user's login handle, the user's bio, and three organisation's name. In github's rest API, you would have to first call the endpoint once for the user, then again for the organisation names.

Advantages
